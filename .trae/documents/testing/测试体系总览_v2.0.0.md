## 1. 测试体系架构

```
┌─────────────────────────────────────────────────────────────┐
│  测试金字塔（按速度/范围分层）                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    E2E（Playwright）                         │
│                 5 个 spec，需三服务齐全                      │
│                  /          \                                │
│              集成测试        真实对话脚本                     │
│          （pytest + mock）  （ws_frame_capture）             │
│             /          \        \                            │
│         单元测试（vitest）  单元测试（pytest）                │
│         前端纯逻辑          后端门禁/工具/MCP                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 层 | 工具 | 位置 | 跑法 | 依赖 |
|----|------|------|------|------|
| 后端单元 | pytest + pytest-asyncio | `apps/backend/tests/` | `cd apps/backend && python -m pytest` | 无（纯逻辑+mock） |
| 前端单元 | vitest + jsdom | `apps/frontend/src/**/*.test.ts` | `cd apps/frontend && npm test` | 无（纯逻辑） |
| 集成 | pytest（httpx + sqlite）| `apps/backend/tests/` | 同上，标记 `@integration` | sqlite（自动） |
| E2E 离线 | Playwright | `apps/frontend/tests/specs/` | `cd tests && npx playwright test` | 前端 dev server |
| E2E @ai | Playwright | 同上（标记 `@ai`）| `RUN_AI_E2E=1 npx playwright test` | 前端 + ZeroClaw daemon |
| 真实对话 | Python 脚本 | `apps/backend/scripts/ws_*.py` | `python scripts/ws_multi_turn_test.py` | ZeroClaw daemon |

## 2. 测试用例编号体系（统一）

格式：`TC-<层>-<模块>-<序号>`

| 前缀 | 层 | 示例 |
|------|---|------|
| `TC-BE-GATE-xxx` | 后端·门禁 | TC-BE-GATE-001 = can_advance_to 禁跨阶段跳 |

```bash
# 后端单元测试（最快，无外部依赖）
cd apps/backend
python -m pytest tests/test_stage_constants.py tests/test_tools_gates.py tests/test_check_gate_structural.py tests/test_mcp_server.py -v

# 前端单元测试
cd apps/frontend
npm test

# Playwright 离线冒烟（需前端 dev server）
cd apps/frontend && npm run dev &
cd tests && npx playwright test zeroclaw-integration --project=chromium

# Playwright @ai 测试（需前端 + ZeroClaw daemon）
RUN_AI_E2E=1 cd tests && npx playwright test zeroclaw-integration

# 真实对话验证（需 ZeroClaw daemon）
cd apps/backend
```
