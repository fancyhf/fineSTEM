# fineSTEM 文档中心

本文档目录统一归档项目的所有技术、产品、运营与规范文档。

## 目录索引

### 1. 规范与标准
- **[api-specs/](./api-specs/)**: API 接口规范与定义。
- **[术语与字典/](./术语与字典/)**: 项目统一术语表与数据字典，作为领域模型的唯一事实来源。

### 2. 设计与架构
- **[技术与架构/](./技术与架构/)**: 技术架构图、部署架构计划、技术选型说明。
- **[产品与规划/](./产品与规划/)**: 产品需求文档 (PRD)、路线图、业务流程设计。
- **[pbl/](./pbl/)**: 项目式学习 (PBL) 的课程设计、教案与评价体系。

### 3. 开发资源
- **[starter_kits/](./starter_kits/)**: 开发启动套件与示例说明。
- **[testing/](./testing/)**: 测试报告与测试计划归档。

### 4. 核心文档导航
- **[项目结构和文档规范.md](./项目结构和文档规范.md)**: 详细定义的项目目录结构与文档管理规则。
- **[创建全面的项目管理规范文档.md](./创建全面的项目管理规范文档（.trae_rules_project_rules.md）.md)**: 项目管理规范草案。
