## 8. 建议优先行动

### 立即行动（1 周内）

1. **接入真实 LLM** — 配置 ZeroClaw Gateway URL，对接 OpenAI/DeepSeek 等真实 LLM API
2. **预置 Demo 数据** — 创建 3 个初始 Demo（含 fork 模板代码），确保 Demo 墙不为空
3. **补充 fork 模板内容** — 为每个 Demo 编写最小可改版代码模板

### 短期行动（2-3 周）

4. 代码版本记录（自动证据采集扩展）
5. AI 对话摘要自动回写为证据
6. 项目包下载完善（含代码文件）
7. PDF/DOCX 排版优化

### 中期行动（4-6 周）

8. WebContainer JS 沙箱集成
9. 扩展页面完善（编辑/删除/详情 + UI 组件化）
10. 问卷引擎多题型 + 填写 UI
11. TrackA/TrackE 路由注册
