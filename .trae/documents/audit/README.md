## 用途

本目录归档 fineSTEM 项目的各类审计产物，包括：

- 架构审计（系统架构、集成链路、协议合规）
- 代码审计（质量、安全、规范合规）
- 合规审计（目录结构、文档归档、环境配置）
- 第三方审计报告（HTML / PDF 等外部产物）

## 范围

- 不存放日常测试报告（见 [../testing/](../testing/)）
- 不存放进度或计划文档（见 [../reports/](../reports/)）
- 仅存放经评审后归档的正式审计结论与证据
