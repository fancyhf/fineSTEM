## 1. 全局布局框架

### 1.1 响应式断点

| 断点名称 | 宽度范围 | 典型设备 | 布局策略 |
|---------|---------|---------|---------|
| `sm` | < 768px | 手机 | 单列，底部导航 |
| `md` | 768px - 1023px | 平板竖屏 | 单列+抽屉，顶部导航 |
| `lg` | 1024px - 1279px | 平板横屏/小笔记本 | 双栏，侧边导航 |
| `xl` | 1280px - 1535px | 标准笔记本 | 三栏可用，侧边导航 |
| `2xl` | >= 1536px | 大屏/桌面 | 最大内容宽度 1440px 居中 |

**主要设计基准**：`xl`（1280px），确保大多数学生笔记本体验最佳。

### 1.2 全局导航

|------|------|------|------|

**收起/展开**：默认收起（64px，仅图标），hover 展开文字（200px）。用户可锁定展开状态。

### 1.3 顶部栏（页面内）

每个页面顶部有一个页面级顶部栏，包含页面标题、搜索、操作按钮：

```text
┌─────────────────────────────────────────────┐
│ 页面标题          [搜索框]       [操作按钮组]  │  h: 56px
└─────────────────────────────────────────────┘
```

### 1.4 通用间距规范

| 用途 | 值 | 说明 |
|------|---|------|
| 页面内边距 | 24px (lg+) / 16px (md/sm) | 主内容区四周 |
| 模块间距 | 32px | 不同功能区之间 |
| 卡片间距 | 16px | 网格卡片之间 |
| 组件内间距 | 16px | 卡片/面板内部 |
| 紧凑内间距 | 8px | 标签/小按钮内部 |
