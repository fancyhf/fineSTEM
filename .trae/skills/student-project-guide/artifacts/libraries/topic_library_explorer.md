# 探索者轨道项目题库 (Explorer Track Topics)

> 适合喜欢尝试新技术、探索未知领域、跨界学习的学生
> 关键词: AI、新技术、实验性、跨学科、前沿

---

## 入门级 (10-12岁, 3-6小时)

### 1. AI绘画提示词生成器
- **描述**: 选择风格、主题、颜色，自动生成AI绘画提示词
- **学习目标**: AI工具使用、提示词工程、创意生成
- **技术栈**: HTML + JavaScript
- **关键词**: AI艺术、提示词工程

### 2. 智能语音助手
- **描述**: 用语音命令控制网页，如"打开计算器"
- **学习目标**: 语音识别API、命令解析、语音反馈
- **技术栈**: HTML + Web Speech API
- **关键词**: 语音交互、智能控制

### 3. AI猜图游戏
- **描述**: 上传图片，AI描述图片内容，用户猜是什么
- **学习目标**: 图像识别API、游戏设计、异步处理
- **技术栈**: HTML + AI Vision API
- **关键词**: 计算机视觉、游戏化

### 4. 智能聊天机器人
- **描述**: 接入AI API，创建可对话的聊天机器人
- **学习目标**: API调用、对话管理、UI设计
- **技术栈**: HTML + JavaScript + AI API
- **关键词**: 对话AI、聊天界面

### 5. 文字转语音朗读器
- **描述**: 输入文字，选择声音，让AI朗读出来
- **学习目标**: TTS技术、音频控制、用户体验
- **技术栈**: HTML + Web Speech API / AI TTS API
- **关键词**: 语音合成、辅助工具

---

## 进阶级 (13-15岁, 6-12小时)

### 6. AI写作助手
- **描述**: 输入主题，AI生成文章大纲和段落，用户编辑完善
- **学习目标**: 文本生成、人机协作、编辑器设计
- **技术栈**: HTML + AI API + 富文本编辑器
- **关键词**: 生成式AI、内容创作

### 7. 智能图片分类器
- **描述**: 上传多张图片，AI自动按内容分类整理
- **学习目标**: 图像识别、批量处理、分类算法
- **技术栈**: Python + AI Vision API
- **关键词**: 图像识别、自动分类

### 8. AI代码解释器
- **描述**: 粘贴代码，AI用简单语言解释代码功能
- **学习目标**: 代码分析、教育设计、API集成
- **技术栈**: HTML + JavaScript + AI API
- **关键词**: 代码教育、AI辅助学习

### 9. 情绪日记分析器
- **描述**: 记录日记，AI分析情绪倾向，生成情绪趋势图
- **学习目标**: 情感分析、数据可视化、隐私保护
- **技术栈**: HTML + AI NLP API + Chart.js
- **关键词**: 情感分析、心理健康

### 10. 智能推荐系统
- **描述**: 根据用户喜好，推荐书籍、电影或音乐
- **学习目标**: 推荐算法、用户画像、协同过滤
- **技术栈**: Python + 简单推荐算法 / AI API
- **关键词**: 推荐系统、个性化

## 挑战级 (16-18岁, 12-20小时)

### 11. 多模态内容生成器
- **描述**: 输入文字描述，同时生成配图、配乐、文案
- **学习目标**: 多模态AI、内容整合、创意工作流
- **技术栈**: Python + 多个AI API整合
- **关键词**: 多模态AI、内容创作

### 12. AI辅助编程环境
- **描述**: 简单的在线代码编辑器，集成AI代码补全和解释
- **学习目标**: 代码编辑器、AI集成、IDE设计
- **技术栈**: HTML + Monaco Editor + AI API
- **关键词**: AI编程、开发工具

### 13. 智能数据清洗工具
- **描述**: 上传CSV，AI自动识别问题并建议清洗方案
- **学习目标**: 数据质量、模式识别、自动化处理
- **技术栈**: Python + Pandas + AI API
- **关键词**: 数据工程、智能自动化

### 14. 虚拟角色对话系统
- **描述**: 创建有特定人设的AI角色，进行角色扮演对话
- **学习目标**: 角色设计、对话系统、提示词工程
- **技术栈**: HTML + AI API + 角色配置
- **关键词**: 角色扮演、对话系统

### 15. AI实验沙盒
- **描述**: 可视化界面测试不同AI模型的能力和限制
- **学习目标**: 模型对比、实验设计、结果分析
- **技术栈**: HTML + 多模型API + 可视化
- **关键词**: AI实验、模型评估

## 选题建议

### 如何从这些题目中选择?

1. **AI兴趣**: 对哪种AI能力最感兴趣? 图像? 文本? 语音?
2. **API获取**: 能否获得所需的AI API访问权限?
3. **伦理意识**: 了解AI使用的伦理边界和隐私保护
4. **人机协作**: 思考如何让AI辅助而非替代人的创造

### AI工具推荐

- **文本生成**: OpenAI GPT, Claude, 文心一言
- **图像生成**: DALL-E, Midjourney, Stable Diffusion
- **语音识别**: Whisper, Web Speech API
- **代码辅助**: GitHub Copilot, CodeT5

### 学习资源

- **提示词工程**: 学习如何与AI有效沟通
- **API文档**: 仔细阅读AI服务的API文档
- **伦理指南**: 了解AI使用的责任边界
