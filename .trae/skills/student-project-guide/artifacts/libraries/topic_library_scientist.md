# 科学家轨道项目题库 (Scientist Track Topics)

> 适合喜欢探索、分析、发现规律的学生
> 关键词: 数据、分析、可视化、模式识别、实验

---

## 入门级 (10-12岁, 3-6小时)

### 1. 我的学习时长追踪器
- **描述**: 记录每天各科目的学习时间，用图表展示
- **学习目标**: 数据收集、基础图表、时间概念
- **技术栈**: HTML + Chart.js
- **关键词**: 数据可视化、习惯养成

### 2. 班级身高分布图
- **描述**: 收集班级同学身高数据，制作柱状图和饼图
- **学习目标**: 数据收集、统计概念、图表类型
- **技术栈**: Python + Matplotlib / HTML + Chart.js
- **关键词**: 统计分析、数据可视化

### 3. 天气日记可视化
- **描述**: 记录一周天气，用颜色编码和图表展示
- **学习目标**: 数据分类、颜色映射、趋势观察
- **技术栈**: HTML + CSS + 简单JS
- **关键词**: 数据记录、可视化设计

### 4. 掷骰子概率实验
- **描述**: 模拟掷骰子，验证概率理论，生成频率分布图
- **学习目标**: 概率概念、随机数、实验验证
- **技术栈**: HTML + JavaScript
- **关键词**: 概率统计、模拟实验

### 5. 我的零花钱记账本
- **描述**: 记录收入和支出，按类别统计，显示结余趋势
- **学习目标**: 数据分类、简单计算、趋势图
- **技术栈**: HTML + JavaScript + LocalStorage
- **关键词**: 财务管理、数据追踪

## 进阶级 (13-15岁, 6-12小时)

### 6. 社交媒体使用分析
- **描述**: 分析自己一周的手机使用数据，生成使用报告
- **学习目标**: 数据导入、时间序列分析、洞察发现
- **技术栈**: Python + Pandas + Matplotlib
- **关键词**: 数字健康、行为分析

### 7. 校园植物识别图鉴
- **描述**: 收集校园植物信息，建立可搜索的数据库
- **学习目标**: 数据建模、搜索功能、分类系统
- **技术栈**: HTML + JavaScript / Python + Flask
- **关键词**: 生物数据、信息检索

### 8. 考试成绩分析器
- **描述**: 输入各科成绩，分析强弱项，生成学习建议
- **学习目标**: 数据分析、条件判断、报告生成
- **技术栈**: Python + Pandas + 可视化库
- **关键词**: 教育数据、个性化分析

### 9. 城市空气质量监测
- **描述**: 获取API数据，展示不同城市的空气质量对比
- **学习目标**: API调用、数据获取、地理可视化
- **技术栈**: HTML + JavaScript + 公共API
- **关键词**: 环境数据、API集成

### 10. 简单问卷调查结果分析
- **描述**: 设计问卷收集数据，自动生成交叉分析报告
- **学习目标**: 问卷设计、数据清洗、关联分析
- **技术栈**: HTML表单 + JavaScript分析
- **关键词**: 调查研究、数据分析

## 挑战级 (16-18岁, 12-20小时)

### 11. 股票数据趋势分析器
- **描述**: 获取股票历史数据，计算技术指标，预测趋势
- **学习目标**: 金融数据、技术分析、预测模型
- **技术栈**: Python + Pandas + yfinance
- **关键词**: 金融分析、时间序列

### 12. 文本情感分析器
- **描述**: 分析电影评论或社交媒体文本的情感倾向
- **学习目标**: NLP基础、情感分析、文本处理
- **技术栈**: Python + NLTK / 调用AI API
- **关键词**: 自然语言处理、机器学习

### 13. 运动数据可视化仪表板
- **描述**: 整合多源运动数据，创建个人健康仪表板
- **学习目标**: 数据整合、仪表板设计、实时更新
- **技术栈**: Python + Dash / HTML + D3.js
- **关键词**: 健康数据、仪表板

### 14. 网络关系图生成器
- **描述**: 分析社交网络或引用关系，生成力导向图
- **学习目标**: 图论基础、网络分析、可视化算法
- **技术栈**: Python + NetworkX + D3.js
- **关键词**: 网络分析、关系可视化

### 15. 科学实验数据记录系统
- **描述**: 为物理/化学实验设计数据记录和分析工具
- **学习目标**: 实验设计、误差分析、科学计算
- **技术栈**: Python + NumPy + Matplotlib
- **关键词**: 科学计算、实验数据

## 选题建议

### 如何从这些题目中选择?

1. **看兴趣**: 对什么数据好奇? 学习? 健康? 环境?
2. **看难度**: 时间预算内能完成吗?
3. **看价值**: 做完后对自己或他人有用吗?
4. **看数据**: 能获取到需要的数据吗?

### 数据获取渠道

- **个人数据**: 手机使用记录、学习时长、消费记录
- **公开API**: 天气、空气质量、股票、地理位置
- **模拟数据**: 用代码生成符合规律的测试数据
- **手动收集**: 问卷、观察记录
