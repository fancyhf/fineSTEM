# Track A: Physics Chaos (Double Pendulum)
> PBL Project ID: `track-a-physics-chaos`

## 项目简介
探索物理世界中的“混沌”现象。通过构建一个交互式的双摆模拟器，直观展示微小的初始条件差异如何导致截然不同的结果。
