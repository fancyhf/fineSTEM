# 创造者轨道项目题库 (Creator Track Topics)

> 适合喜欢动手制作、解决实际问题、创造产品的学生
> 关键词: 工具、应用、游戏、创意、实用

---

## 入门级 (10-12岁, 3-6小时)

### 1. 我的待办清单
- **描述**: 可以添加、完成、删除任务的简单清单应用
- **学习目标**: DOM操作、事件处理、数据存储
- **技术栈**: HTML + CSS + JavaScript
- **关键词**: 任务管理、CRUD操作

### 2. 简易计算器
- **描述**: 支持加减乘除的计算器，带历史记录
- **学习目标**: 事件监听、状态管理、计算逻辑
- **技术栈**: HTML + CSS + JavaScript
- **关键词**: 交互设计、数学运算

### 3. 幸运转盘
- **描述**: 可自定义选项的转盘，用于抽签或决策
- **学习目标**: Canvas绘图、动画、随机数
- **技术栈**: HTML + Canvas API
- **关键词**: 游戏化、动画效果

### 4. 个人名片生成器
- **描述**: 输入信息生成漂亮的个人名片图片
- **学习目标**: 表单处理、样式生成、图片导出
- **技术栈**: HTML + CSS + html2canvas
- **关键词**: 设计工具、内容生成

### 5. 倒计时器
- **描述**: 可设置多个倒计时的工具，带提醒功能
- **学习目标**: 定时器、状态管理、通知API
- **技术栈**: HTML + JavaScript
- **关键词**: 时间管理、实用工具

## 进阶级 (13-15岁, 6-12小时)

### 6. 在线画板
- **描述**: 支持画笔、橡皮擦、颜色选择、保存作品的画板
- **学习目标**: Canvas绘图、鼠标事件、文件操作
- **技术栈**: HTML + Canvas API + File API
- **关键词**: 创意工具、绘图应用

### 7. 单词记忆卡片
- **描述**: 正面显示单词，背面显示释义，支持翻卡和测试
- **学习目标**: 卡片翻转动画、数据管理、测试逻辑
- **技术栈**: HTML + CSS3 + JavaScript
- **关键词**: 教育工具、记忆训练

### 8. 简易音乐播放器
- **描述**: 播放本地音乐文件，显示播放列表和进度
- **学习目标**: 音频API、播放控制、列表管理
- **技术栈**: HTML + Web Audio API
- **关键词**: 媒体播放、UI设计

### 9. 密码生成器
- **描述**: 根据规则生成安全密码，评估密码强度
- **学习目标**: 随机数生成、字符串操作、安全概念
- **技术栈**: HTML + JavaScript
- **关键词**: 安全工具、实用程序

### 10. 响应式个人简历页
- **描述**: 制作可在手机和电脑上查看的个人简历网页
- **学习目标**: 响应式设计、布局技巧、内容展示
- **技术栈**: HTML + CSS (Flexbox/Grid)
- **关键词**: 网页设计、响应式布局

## 挑战级 (16-18岁, 12-20小时)

### 11. 多人协作白板
- **描述**: 多人实时协作的在线白板，支持画笔和便签
- **学习目标**: WebSocket、实时同步、并发处理
- **技术栈**: HTML + WebSocket + Node.js/Flask
- **关键词**: 实时协作、网络编程

### 12. Markdown笔记应用
- **描述**: 支持Markdown编辑、实时预览、本地存储的笔记工具
- **学习目标**: Markdown解析、编辑器集成、数据持久化
- **技术栈**: HTML + marked.js + LocalStorage/IndexedDB
- **关键词**: 内容编辑、本地应用

### 13. 简易游戏引擎
- **描述**: 制作可配置的游戏框架，支持精灵、碰撞、场景
- **学习目标**: 游戏循环、物理引擎、架构设计
- **技术栈**: HTML + Canvas + JavaScript
- **关键词**: 游戏开发、架构设计

### 14. 网站内容抓取器
- **描述**: 输入网址，提取文章标题、正文、图片
- **学习目标**: HTTP请求、HTML解析、数据提取
- **技术栈**: Python + requests + BeautifulSoup
- **关键词**: 网络爬虫、数据提取

### 15. 自动化脚本工具箱
- **描述**: 批量重命名文件、整理文件夹、生成报告等工具集合
- **学习目标**: 文件系统操作、批处理、自动化
- **技术栈**: Python + os/pathlib + tkinter
- **关键词**: 自动化、效率工具

## 选题建议

### 如何从这些题目中选择?

1. **解决痛点**: 你现在有什么想解决的问题?
2. **使用频率**: 做完后会经常使用吗?
3. **展示价值**: 可以向别人展示成果吗?
4. **技术挑战**: 能学到想学的技术吗?

### 功能扩展思路

每个项目都可以考虑这些扩展:
- **数据持久化**: 保存到本地存储或云端
- **分享功能**: 导出图片/链接分享给别人
- **主题切换**: 支持深色/浅色模式
- **移动端适配**: 在手机上也能良好使用
