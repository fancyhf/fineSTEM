# 设计文档 - 奇幻选择之旅

## 技术方案
- **方案**: HTML + CSS + JavaScript（纯前端）
- **运行方式**: 浏览器直接打开，零安装

## UI设计
- **风格**: 复古像素风（棕色/绿色调）
- **布局**: 居中竖排（RPG对话框风格）

## 功能元素
1. 标题区（游戏名称 + 进度条 + 章节标签）
2. 配图区（Canvas像素画）
3. 故事区（叙事文本）
4. 选项区（选择题按钮）
5. 状态区（结局计数 + 重新开始）

## 验收条件
1. ✅ 至少3个故事章节
2. ✅ 配图用像素画（Canvas绘制）
3. ✅ 每个章节≥2个选择
4. ✅ 有进度条显示
5. ✅ 有不同结局（至少2个）

## 数据流
```
加载页面 → 显示序章 → 展示故事+配图+选项
    ↓
玩家点击选项 → 跳转下一章节 → 更新故事+配图+选项
    ↓
到达结局 → 记录结局 → 显示结局画面 + 可重新开始
```
