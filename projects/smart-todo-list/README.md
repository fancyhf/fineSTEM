## 项目介绍

自动排序优先级的待办App，AI帮你决定先做什么。

这是一个使用 Next.js + React + TypeScript 开发的智能待办清单应用，适合初中生学习 Web 开发。

## 规划功能

- 任务管理: 添加、编辑、删除任务
- 智能排序: AI 根据优先级自动排序
- 优先级可视化: 直观展示任务紧急程度
- 本地存储: 数据保存在浏览器中

## 技术栈

- 框架: Next.js
- 前端: React + TypeScript
- 样式: CSS Modules / Tailwind CSS
- 存储: LocalStorage
