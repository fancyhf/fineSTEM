# 静态资源目录 (static/)

本目录存放项目的静态资源文件，包括 CSS 样式和 JavaScript 脚本。

## 目录结构

```
static/
├── css/
│   └── style.css       # 主样式文件 (卡哇伊二次元风格)
└── js/
    └── main.js         # 主 JavaScript 文件
```

## 文件说明

### css/style.css

主样式文件，定义了：

- 卡哇伊二次元风格配色
- 卡片翻转动画效果
- 响应式布局
- 按钮和表单样式

### js/main.js

主 JavaScript 文件，实现：

- 卡片翻转交互
- 搜索功能
- 收藏切换
- 分类筛选
- AJAX 请求处理

## 样式特点

- 主色调: 粉色系、紫色系
- 圆角设计
- 阴影效果
- 动画过渡
