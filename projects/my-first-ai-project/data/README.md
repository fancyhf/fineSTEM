# 数据目录 (data/)

本目录存放项目的数据文件。

## 文件说明

| 文件 | 说明 |
|------|------|
| poetry_data.json | 古诗词数据存储文件 |

## 数据格式

poetry_data.json 采用 JSON 格式存储诗词数据，每条记录包含：

- `id`: 唯一标识
- `title`: 诗词标题
- `author`: 作者
- `dynasty`: 朝代
- `content`: 诗词内容
- `category`: 分类 (唐诗/宋词/元曲等)
- `tags`: 标签

## 注意事项

- 数据文件为 UTF-8 编码
- 修改数据后需重启应用生效
- 建议定期备份数据文件
