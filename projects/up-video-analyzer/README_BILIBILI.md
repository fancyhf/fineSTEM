# 📺 B 站视频字幕导入使用说明

## 🎉 新功能上线！

现在你可以直接从 B 站视频链接提取字幕，无需手动下载！

## 🚀 快速开始

### 步骤 1：复制视频链接
在浏览器中打开任意 B 站视频，复制地址栏 URL。

例如：
```
https://www.bilibili.com/video/BV1GJ411x7h7
```

### 步骤 2：粘贴到应用
1. 打开应用：http://localhost:8501
2. 选择"🔗 粘贴视频链接"选项
3. 将链接粘贴到输入框

### 步骤 3：提取字幕
点击"🔄 提取 B 站字幕"按钮，等待几秒钟。

### 步骤 4：开始分析
提取成功后，点击"🔍 开始分析"即可查看：
- ☁️ 词频可视化图表
- 📊 数据统计面板
- 🤖 AI 智能总结
- 📥 导出分析报告

## ✅ 成功示例

如果视频有字幕，你会看到：
```
✅ 成功提取《视频标题》的字幕（中文）
共 1234 字符
```

## ❌ 失败处理

如果视频没有字幕，你会看到提示信息：
```
ℹ️ 视频《XXX》没有找到字幕

可能的原因：
• UP 主未上传字幕
• 视频没有自动生成字幕
• 字幕权限受限

💡 建议：
1. 使用「唧唧」等工具下载字幕
2. 手动复制视频简介或评论
3. 联系 UP 主提供字幕
```

## 💡 提示

1. **不是所有视频都有字幕** - B 站很多视频没有字幕功能
2. **优先选择有 CC 字幕的视频** - UP 主上传的字幕最可靠
3. **短链接也支持** - b23.tv 开头的短链接同样可用
4. **av 号/BV 号都支持** - 两种格式的视频 ID 都可以

## 🎯 推荐测试视频

想找有字幕的视频测试？推荐：
- 教育类视频（通常有字幕）
- 官方账号发布的视频
- 带有"CC 字幕"标识的视频

## 🔧 故障排除

### 问题 1：提示"无法获取视频信息"
**解决**：检查链接是否正确，或视频是否已被删除

### 问题 2：一直加载中
**解决**：检查网络连接，或稍后再试

### 问题 3：提取成功但内容为空
**解决**：该视频可能只有字幕文件但没有实际内容

## 📞 需要帮助？

如果遇到问题：
1. 查看详细日志
2. 尝试其他视频
3. 使用手动上传方式（上传 SRT/TXT 文件）
