# 📺 UP 主视频内容分析器 - 完整使用指南

## 🎯 项目概述

这是一个用于分析视频内容的工具，支持：
- ✅ 词频统计和可视化
- ✅ 数据统计面板
- ✅ AI 智能总结
- ✅ 报告导出功能

## 📋 功能说明

### 1. 上传字幕文件（推荐）
- 支持 SRT 字幕文件
- 支持 TXT 纯文本文件
- 100% 可靠，长期稳定

### 2. 视频链接粘贴（暂停）
- 由于 B 站 API 限制，此功能暂时不可用
- 可以手动获取视频信息

## 🚀 快速开始

### 方法一：手动上传（推荐）

#### 步骤 1：准备字幕文件
1. **从 B 站获取字幕**
   - 使用 [DownKyi](https://github.com/leiurayer/downkyi) 下载
   - 使用 Bilibili-Evolved 浏览器扩展
   - 手动复制视频简介或评论

2. **保存为文件**
   - 保存为 `.srt` 或 `.txt` 格式
   - 确保编码为 UTF-8

#### 步骤 2：上传分析
1. 打开应用：`http://localhost:8501`
2. 选择"📁 上传字幕文件"
3. 选择准备好的文件
4. 点击"🔍 开始分析"
5. 查看结果

### 方法二：视频链接（暂停）
- 此功能因 API 限制暂时不可用
- 建议使用方法一

## 📝 详细使用流程

### 1. 获取字幕文件

#### 方案 A：使用 DownKyi（推荐）
```
1. 下载并安装 DownKyi
2. 复制 B 站视频链接
3. 粘贴到 DownKyi
4. 选择下载字幕
5. 导出为 SRT 或 TXT
6. 上传到分析器
```

#### 方案 B：使用浏览器扩展
```
1. 安装 Bilibili-Evolved
2. 打开目标视频
3. 点击扩展图标
4. 选择下载字幕
5. 保存为文本文件
6. 上传到分析器
```

#### 方案 C：手动复制
```
1. 打开视频简介
2. 复制文字内容
3. 保存为 TXT 文件
4. 上传到分析器
```

### 2. 上传分析
```
1. 选择"📁 上传字幕文件"
2. 拖拽或点击选择文件
3. 点击"🔍 开始分析"
4. 等待处理完成
5. 查看结果
```

## 🛠️ 技术支持

### 文件格式要求
- **SRT**: 标准字幕格式
- **TXT**: 纯文本格式
- **编码**: UTF-8
- **大小**: 无限制

### 内容要求
- 中文内容优先
- 有效文本为主
- 避免过多标点符号

## ⚠️ 常见问题

### Q: 自动提取功能什么时候恢复？
A: 目前没有确切时间，因为 B 站 API 限制较严格。

### Q: 为什么推荐手动上传？
A: 
- 更可靠（100% 成功率）
- 更灵活（可预处理）
- 更稳定（不受 API 变化影响）

### Q: 如何获得更好的分析效果？
A: 
- 提供更高质量的文本
- 去除无关内容
- 确保文本连续性

### Q: 支持其他视频平台吗？
A: 目前主要支持 B 站，其他平台可通过手动上传方式使用。
