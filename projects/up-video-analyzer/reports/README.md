# 报告目录 (reports/)

本目录存放项目的测试报告和截图。

## 文件说明

| 文件 | 说明 |
|------|------|
| test_report.json | 测试报告 (JSON 格式) |
| 01_page_load.png | 页面加载截图 |
| 02_ui_elements.png | UI 元素截图 |
| 03_url_confirmed.png | URL 确认截图 |
| 03_url_input.png | URL 输入截图 |
| 04_task_history.png | 任务历史截图 |
| 05_report_tab.png | 报告标签页截图 |
| 06_upload_section.png | 上传区域截图 |

## 报告用途

这些报告和截图可用于：

- 项目验收证据
- 功能演示文档
- 问题排查参考
- 教学案例展示

## 测试报告格式

test_report.json 包含：
- 测试时间
- 测试用例列表
- 通过/失败状态
- 错误信息 (如有)
