# B 站视频字幕导入指南

## 🎯 功能说明

本应用现已支持**直接从 B 站视频链接提取字幕**，无需手动下载！

## 📝 使用步骤

### 方法一：自动提取（推荐）

1. **复制 B 站视频链接**
   - 在浏览器中打开 B 站视频
   - 复制地址栏 URL（例如：`https://www.bilibili.com/video/BV1xx411c7mD`）

2. **粘贴到应用**
   - 在应用中选择"🔗 粘贴视频链接"
   - 粘贴链接到输入框
   - 点击"🔄 提取 B 站字幕"按钮

3. **等待提取完成**
   - 系统会自动连接 B 站 API
   - 提取成功后显示字幕内容
   - 点击"🔍 开始分析"即可查看词频统计

### 方法二：手动上传

如果自动提取失败，可以手动获取字幕：

#### 使用 DownKyi（唧唧）
1. 下载并安装 [DownKyi](https://github.com/leiurayer/downkyi)
2. 粘贴视频链接下载
3. 导出字幕为 SRT 或 TXT 格式
4. 上传到应用进行分析

#### 使用浏览器扩展
1. 安装 B 站字幕下载扩展（如"B 站字幕助手"）
2. 在视频页面点击扩展图标
3. 下载字幕文件
4. 上传到应用

## ⚠️ 注意事项

### 支持的字幕类型
- ✅ UP 主上传的 CC 字幕
- ✅ 视频自动生成字幕
- ✅ 用户投稿字幕

### 可能失败的原因
- ❌ 视频没有字幕（最常见）
- ❌ 字幕权限受限
- ❌ 网络问题导致 API 请求失败
- ❌ 视频链接格式不正确

### 正确的链接格式
- ✅ `https://www.bilibili.com/video/BV1xx411c7mD`
- ✅ `https://b23.tv/xxxxx`（短链接）
- ✅ `https://www.bilibili.com/video/av12345678`（av 号）

## 💡 常见问题

### Q: 提示"没有找到字幕"怎么办？
A: 说明该视频确实没有字幕。可以尝试：
- 联系 UP 主请求提供字幕
- 使用视频的文字简介或评论
- 手动整理视频内容

### Q: 提取失败怎么办？
A: 检查以下几点：
1. 确认视频链接是否正确
2. 确认视频是否有字幕
3. 检查网络连接
4. 尝试使用手动上传方式

### Q: 支持其他平台吗？
A: 目前仅支持 B 站。抖音、西瓜视频等平台由于 API 限制，暂时不支持自动提取。

## 🔧 技术原理

应用通过以下方式提取字幕：
1. 解析视频链接获取 BV 号/av 号
2. 调用 B 站 API 获取视频 cid
3. 查询字幕列表
4. 下载并解析字幕文件
5. 提取纯文本内容

整个过程无需登录，使用公开 API 接口。
