❓ 6. 萌新常见问题

Q1: 为什么要用 try-except？
A: 防止程序出错就崩溃。就像开车系安全带。
Q2: **kwargs 是什么？
A: Python 的特殊语法，表示"任意数量的关键字参数"。
Q3: 为什么要用 with 语句？
A: with 会自动管理资源。
Q4: Optional[str] 是什么？
A: Python 的类型提示。
Q5: @dataclass 有什么用？
A: 自动生成类的初始化方法。
