# ⚠️ B 站字幕提取功能说明

## 🚨 重要提示

**由于 B 站 API 限制，自动提取功能目前无法使用。**

---

## ❌ 问题原因

B 站从 2024 年开始实施了更严格的 API 访问控制：

1. **WBI 签名加密** - 所有 API 请求都需要复杂的签名
2. **Cookie 验证** - 需要登录状态才能访问
3. **频率限制** - 频繁请求会被封禁
4. **跨域限制** - 第三方应用无法直接访问

---

## ✅ 可行的替代方案

### 方案 1：手动上传字幕文件（推荐）

#### 步骤 1：下载字幕
使用以下工具下载字幕：

**DownKyi（唧唧）**
- 下载地址：https://github.com/leiurayer/downkyi
- 使用方法：
  1. 粘贴视频链接
  2. 选择下载字幕
  3. 导出为 SRT 或 TXT

**Bilibili-Evolved（浏览器扩展）**
- 安装地址：Chrome/Edge 扩展商店
- 使用方法：
  1. 打开 B 站视频
  2. 点击扩展图标
  3. 选择"下载字幕"

#### 步骤 2：上传到应用
1. 打开应用：http://localhost:8501
2. 选择"📁 上传字幕文件"
3. 上传 SRT 或 TXT 文件
4. 点击"🔍 开始分析"

## 📝 手动上传步骤

### 准备字幕文件

**格式要求**：
- ✅ SRT 格式（标准字幕格式）
- ✅ TXT 格式（纯文本）
- ✅ 编码：UTF-8

**文件内容**：
- 可以是字幕文件
- 可以是视频简介
- 可以是评论区内容
- 可以是手动整理的文字

### 上传分析

1. **打开应用**
   ```
   http://localhost:8501
   ```

2. **选择上传方式**
   - 点击"📁 上传字幕文件"

3. **上传文件**
   - 拖拽文件到上传区域
   - 或点击选择文件

4. **开始分析**
   - 点击"🔍 开始分析"
   - 查看词频统计
   - 查看 AI 总结

5. **导出结果**
   - 下载词频图表（PNG）
   - 下载词频数据（CSV）

## 🔧 技术说明

### 为什么自动提取不可用？

**B 站 API 限制**：
- 需要 WBI 签名（动态密钥）
- 需要 Cookie（登录状态）
- 需要 Referer 验证
- 有访问频率限制

**实现复杂度**：
- 需要模拟浏览器
- 需要维护 Cookie
- 需要处理加密
- 需要定期更新

**成本考虑**：
- 开发成本高
- 维护成本高
- 稳定性差
- 可能被封禁

### 为什么手动上传更可靠？

**优点**：
- ✅ 100% 可靠
- ✅ 不受 API 限制
- ✅ 可以预处理
- ✅ 可以自定义
- ✅ 长期稳定

**效率**：
- 下载字幕：1-2 分钟
- 上传分析：10 秒
- 总计：2-3 分钟
