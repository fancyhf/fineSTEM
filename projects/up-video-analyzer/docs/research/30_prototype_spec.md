## 3. 交互流程

### 3.1 字幕文件上传流程

```
用户选择"上传字幕文件"
    ↓
用户拖拽/选择 SRT/TXT 文件
    ↓
系统自动解析字幕内容
    ↓
显示解析结果（字符数、行数）
    ↓
用户点击"开始分析"
    ↓
系统执行：分词 → 过滤 → 统计 → 词云
    ↓
展示分析结果（词云图 + 统计面板）
    ↓
用户可选择"导出报告"
```

### 3.2 视频链接分析流程

```
用户选择"粘贴视频链接"
    ↓
用户在 textarea 输入链接
    ↓
用户点击"确认链接"
    ↓
系统检测平台（B站/抖音/YouTube）
    ↓
显示平台信息 + 提取选项
    ↓
用户选择"提取字幕"或"下载视频提取"
    ↓
系统执行任务（异步）
    ↓
任务完成后自动显示结果
    ↓
结果保存到任务历史
```

### 3.3 任务历史查看流程

```
用户点击"任务历史"标签
    ↓
加载并显示所有历史任务
    ↓
用户可查看任务详情/删除任务/重新分析
    ↓
点击"查看报告"跳转到执行报告页
```

## 4. 状态管理

### 4.1 Session State 变量

| 变量名 | 类型 | 用途 |
|--------|------|------|
| active_tab | str | 当前活动标签页 |
| confirmed_video_url | str | 已确认的视频链接 |
| confirmed_platform | str | 已检测的平台 |
| extracted_content | str | 提取的字幕内容 |
| current_task_id | str | 当前任务ID |
| reanalyze_task_id | str | 重新分析的任务ID |

### 4.2 任务状态流转

```
PENDING (等待中)
    ↓ 开始下载
DOWNLOADING (下载中)
    ↓ 下载完成
EXTRACTING (提取中)
    ↓ 提取完成
COMPLETED (已完成) ←→ FAILED (失败)
```

## 5. 样式规范

### 5.1 颜色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色调 | 绿色 | #4CAF50 |
| 背景色 | 白色 | #FFFFFF |
| 边框色 | 灰色 | #E0E0E0 |

### 5.2 布局规范

- 页面最大宽度: 1200px
- 左右分栏比例: 2:1

- 按钮圆角: 8px
