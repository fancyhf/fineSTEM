# 研学文档目录 (research/)

本目录存放项目的研学文档，适合用于研学报告、项目展示和学习总结。

## 文档列表

| 文档 | 说明 | 用途 |
|------|------|------|
| 10_proposal.md | 开题报告 | 项目开题答辩 |
| 20_prd_design.md | 需求与设计文档 | 产品需求说明 |
| 30_prototype_spec.md | 原型设计说明书 | 原型设计说明 |
| 40_tech_report.md | 技术报告 | 技术实现记录 |
| 50_final_report.md | 结题报告 | 项目总结报告 |
| 60_paper.md | 论文 | 学术论文格式 |

## 文档使用指南

### 开题阶段
- 参考 [10_proposal.md](./10_proposal.md) 了解研究背景和问题

### 设计阶段
- 参考 [20_prd_design.md](./20_prd_design.md) 了解需求设计
- 参考 [30_prototype_spec.md](./30_prototype_spec.md) 了解原型设计

### 实现阶段
- 参考 [40_tech_report.md](./40_tech_report.md) 了解技术实现

### 结题阶段
- 参考 [50_final_report.md](./50_final_report.md) 了解项目总结
- 参考 [60_paper.md](./60_paper.md) 了解论文格式

## 写作建议

### 对于学生
1. 按顺序阅读各文档，了解完整的项目流程
2. 参考文档结构，撰写自己的研学报告
3. 注意记录学习过程中的心得体会

### 对于老师
1. 可将这些文档作为教学案例
2. 根据学生水平调整文档要求
3. 评估学生文档的完整性和质量
