## 5. 字幕处理流程图

```mermaid
flowchart TD
    A[输入字幕] --> B{输入类型}
    
    B -->|SRT 文件| C[解析 SRT 格式]
    B -->|TXT 文件| D[直接读取文本]
    B -->|视频链接| E[提取字幕]
    
    C --> F[提取时间轴]
    C --> G[提取文本内容]
    
    E --> H{平台检测}
    H -->|B站| I[调用 B站 API]
    H -->|YouTube| J[调用 YouTube API]
    H -->|其他| K[下载视频]
    
    I --> L[获取字幕]
    J --> L
    K --> M[语音识别]
    M --> N[SiliconFlow API]
    N --> L
    
    F --> O[合并文本]
    G --> O
    D --> O
    L --> O
    
    O --> P[文本预处理]
    P --> Q[去除特殊字符]
    Q --> R[统一编码格式]
    
    R --> S[中文分词]
    S --> T[jieba.cut]
    T --> U[停用词过滤]
    
    U --> V{停用词库}
    V -->|是停用词| W[过滤]
    V -->|不是停用词| X[保留]
    
    W --> Y[词频统计]
    X --> Y
    
    Y --> Z[Counter 统计]
    Z --> AA[排序 TopN]
    
    AA --> AB[输出结果]
    AB --> AC[词频字典]
    
    style A fill:#4CAF50,color:#fff
    style AB fill:#2196F3,color:#fff
    style Y fill:#FF9800,color:#fff
```

## 6. 用户交互流程图

```mermaid
sequenceDiagram
    actor 用户
    participant UI as Web界面
    participant TM as 任务管理器
    participant SP as 字幕处理器
    participant AN as 分析引擎
    participant DB as 数据存储
    
    用户->>UI: 访问应用
    UI->>TM: 获取任务列表
    TM->>DB: 读取任务数据
    DB-->>TM: 返回任务列表
    TM-->>UI: 显示任务历史
    UI-->>用户: 展示界面
    
    alt 上传字幕文件
        用户->>UI: 选择 SRT/TXT 文件
        UI->>SP: 上传文件
        SP->>SP: 解析字幕
        SP->>AN: 发送文本
    else 输入视频链接
        用户->>UI: 输入视频 URL
        UI->>TM: 创建任务
        TM->>DB: 保存任务
        TM->>SP: 提取字幕
        SP->>SP: 平台检测
        SP->>SP: 下载/提取
        SP->>AN: 发送文本
    end
    
    AN->>AN: 中文分词
    AN->>AN: 停用词过滤
    AN->>AN: 词频统计
    AN->>AN: 生成词云
    AN-->>TM: 返回结果
    TM->>DB: 保存结果
    TM-->>UI: 更新状态
    UI-->>用户: 展示分析结果
    
    用户->>UI: 查看词云
    用户->>UI: 查看统计
    用户->>UI: 导出报告
    UI->>DB: 读取结果文件
    DB-->>UI: 返回文件
    UI-->>用户: 下载报告
```
