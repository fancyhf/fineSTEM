## ⚠️ 重要说明

### 为什么有些视频提取失败？

**根本原因**：**不是所有 B 站视频都有字幕！**

B 站视频的字幕分为三种：
1. **UP 主上传的 CC 字幕** - ✅ 可以提取
2. **AI 自动生成字幕** - ✅ 可以提取
3. **没有字幕** - ❌ 无法提取

### 如何判断视频是否有字幕？

在 B 站视频页面，查看：
- 视频标题下方是否有 **"CC" 图标**
- 视频播放器右下角是否有 **字幕选项**
- 视频简介中是否提到字幕

## 💡 如果提取失败怎么办？

### 方案 1：换一个有字幕的视频
- 寻找有 **CC 标识** 的视频
- 选择教育、演讲、新闻类视频
- 避免娱乐短视频

### 方案 2：手动上传字幕文件
1. 使用 [DownKyi](https://github.com/leiurayer/downkyi) 下载字幕
2. 使用 Bilibili-Evolved 浏览器扩展
3. 手动复制视频简介内容
4. 上传 SRT/TXT 文件到应用

### 方案 3：使用其他工具生成字幕
- **剪映**：导入视频，自动生成字幕
- **讯飞听见**：语音转文字
- **网易见外**：在线字幕生成

## 🔧 技术实现

### 原理
```
1. 访问视频页面
2. 提取视频 ID（BV 号/av 号）
3. 调用 B 站 API 获取 cid
4. 查询字幕列表
5. 下载字幕 JSON
6. 提取纯文本内容
```

### 文件位置
- 字幕提取模块：`src/bilibili_subtitle_v3.py`
- 主程序：`src/main.py`

## 📊 常见问题

### Q: 提示"没有找到字幕"？
A: 这个视频确实没有字幕，换一个有 CC 标识的视频。

### Q: 提示"网络超时"？
A: 检查网络连接，或稍后再试。

### Q: 提示"数据格式错误"？
A: B 站 API 可能更新了，需要更新代码。

### Q: 所有视频都提取失败？
A: 
1. 检查网络连接
2. 尝试使用 VPN
3. 使用手动上传方式
