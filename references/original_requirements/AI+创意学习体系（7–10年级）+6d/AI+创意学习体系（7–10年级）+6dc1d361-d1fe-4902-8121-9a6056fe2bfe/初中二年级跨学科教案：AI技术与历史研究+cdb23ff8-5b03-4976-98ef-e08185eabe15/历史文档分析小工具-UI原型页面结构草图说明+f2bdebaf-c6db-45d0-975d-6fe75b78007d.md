
整个产品共有 **五个主要界面**：

1. 首页（文档库）

2. 上传弹窗

3. 文档概览页

4. 详细页签（词频 / 时间轴 / 地图 / 数据）

5. 通用侧边栏与提示层

---

## 🏠 1. 首页（文档库）

### 页面结构

```Plain Text
┌────────────────────────────┐
│ 顶部导航栏：LOGO｜历史文档分析工具｜上传文档按钮 │
├────────────────────────────┤
│ 搜索栏 + 状态筛选下拉 + 排序按钮                    │
├────────────────────────────┤
│ 文档列表（卡片或表格形式）                         │
│ ┌────────────────────────┐ │
│ │标题：马可波罗游记        状态：分析完成 ✅           │
│ │页数：312    上传时间：2025-11-03  │
│ │人物数：128  地点数：42  事件数：57 │
│ │[查看详情] [导出结果] [删除] │
│ └────────────────────────┘ │
│ ...多条文档...                                     │
├────────────────────────────┤
│ 空状态插画 + 文案：“还没有文档。上传你的第一份资料吧。” │
└────────────────────────────┘
```

### 交互行为

- 点击「上传文档」→ 打开上传弹窗。

- 点击文档卡片 → 进入“文档概览页”。

- 状态筛选（全部 / 分析中 / 已完成 / 失败）。

- 搜索栏即时过滤。

- Hover 状态下显示操作菜单（重新分析 / 删除）。

## ⬆️ 2. 上传弹窗

### 布局结构

```Plain Text
┌───────────────────────────┐
│ 📄 上传文档                                                   │
├───────────────────────────┤
│ [ 拖拽或点击上传框 ]                                         │
│  支持：PDF / DOCX / TXT                                      │
│  文件名自动填入标题栏                                        │
│  语言检测结果：中文（可修改）                                │
│──────────────────────────│
│ 标题输入框 [ 马可波罗游记 ]                                  │
│ 语言选择框 [ 中文 ▼ ]                                        │
│──────────────────────────│
│ [ 上传并开始分析 ] 按钮                                      │
│ 进度条：文本提取 → 事件识别 → 时间轴生成 → 地图绘制          │
└───────────────────────────┘
```

### 行为

- 上传成功 → 提示“已开始分析，可在首页查看进度”。

- 上传错误 → 提示“文件格式不支持或过大”。

- 上传中显示动态进度条与状态文字。

## 📎 5. 通用侧边栏与提示层

- **证据侧栏**：右侧滑出，展示某项对应的原文句子与页码，带复制按钮。

- **状态提示层**：顶部小条提示分析进度 / 成功 / 失败。

- **空状态页**：卡通插画 + 说明文字 + “重新上传”按钮。

- **加载骨架屏**：卡片灰底动画，用于词频榜、地图、表格加载时。

---

## 🎯 交互一致性与视觉基调建议

- 主色调：温和的中性色（浅褐/青灰），符合“历史资料”氛围。

- 视觉风格：卡片分区清晰，字号偏大（中学生友好）。

- 所有图表与地图 hover 时高亮并显示说明卡。

- 页面切换采用滑入动画（保持学习节奏感）。
