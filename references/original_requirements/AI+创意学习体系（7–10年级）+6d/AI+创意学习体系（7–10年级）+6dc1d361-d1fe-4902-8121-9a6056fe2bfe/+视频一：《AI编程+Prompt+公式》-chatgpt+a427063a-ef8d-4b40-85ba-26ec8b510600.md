## 🧩 第三幕：讲公式（核心教学）

**标题：AI 编程 Prompt 公式**

> **Prompt = 角色 + 任务 + 输入 + 输出格式 + 限制条件**

**示范讲解：**





## 示例 1：终端小游戏《石头剪刀布》（带比分）

**要点**：贴近游戏兴趣；看到“能玩起来”的代码。
**可复制 Prompt：**

> 角色/背景：你是一名 Python 编程助教。
任务目标：写一个“石头剪刀布”终端小游戏，玩家 vs 电脑，先到 3 分获胜。
输入数据：玩家从输入中键入“石/剪/布”。
输出格式：只输出可运行的 Python 代码，不要解释。每回合打印双方选择、胜负、比分。
限制条件：使用 `random` 标准库；结构清晰、变量名可读。
测试用例：玩家依次输入“石 回车 剪 回车 布 回车 …”能正常计分并结束游戏。

（演示：AI吐出代码→复制运行→三回合就能看到比分增长）

## 示例 2：单词测验生成器（自带答案核对）

**要点**：学习“赋能课业”，但不枯燥；能立即用于英语/学科。
**可复制 Prompt：**

> 角色/背景：你是 Python 编程老师。
任务目标：生成一个“英语单词测验脚本”。程序随机抽取给定词表的 5 个单词，给出中文释义，要求用户输入英文；最后统计正确数并显示答案对照。
输入数据：词表 = `[("biology","生物"),("planet","行星"),("oxygen","氧气"),("gravity","重力"),("experiment","实验"),("cell","细胞")]`
输出格式：只输出可运行的 Python 代码，不要解释。
限制条件：不得使用第三方库；题目顺序随机；用户输入忽略大小写与两端空格。
测试用例：把上面的词表直接粘进代码能运行；答题后会看到“你的得分：x/5”。

（演示：运行→答几题→显示得分与正确答案）

## 示例 3：表情符号翻译器（把句子变“梗”）

**要点**：有趣、社交属性强，立刻想分享。
**可复制 Prompt：**

> 角色/背景：你是 Python 初学者的助教。
任务目标：写一个“表情符号翻译器”，把句子里常见词替换成对应 emoji，其他词保持原样。
输入数据：内置一个词典，例如：{"开心":"😄","生气":"😡","学习":"📚","作业":"📝","篮球":"🏀","音乐":"🎵"}；用户输入任意中文句子。
输出格式：只输出可运行的 Python 代码；运行后先打印原句，再打印替换后的句子。
限制条件：不使用第三方库；替换需按词匹配而非逐字；保留标点。
测试用例：输入“今天学习完作业去打篮球，心情很开心！”应出现多处 emoji。

（演示：输入一句话→输出带 emoji 的版本）

## 黑白对比：坏 Prompt vs. 好 Prompt

- **坏**：`帮我写个游戏代码。`（信息太少，AI乱猜）

- **好**：包含**角色/背景、任务目标、输入数据、输出格式、限制条件、测试用例**（AI按你逻辑生成）

”
